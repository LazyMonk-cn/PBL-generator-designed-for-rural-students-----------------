# 乡村儿童学习助手

## 项目简介
这是一个专为乡村儿童教育设计的AI助手项目，旨在通过项目式学习(PBL)和AI技术，为乡村地区的教师和学生提供优质的教育资源和学习支持。

## 项目结构
```
src/
├── index.html              # 主页面
├── css/
│   └── styles.css          # 所有样式文件
├── js/
│   └── main.js             # 所有JavaScript逻辑
└── assets/                 # 静态资源文件（图片等）
```

## 功能模块

### 1. PBL任务生成器
- 为乡村儿童创建项目式学习任务
- 支持按年级和主题生成个性化任务
- 结合乡村实际生活场景，简单易行

### 2. 学习资源推荐
- 根据学生兴趣方向推荐学习资源
- 资源适合乡村环境，考虑资源有限的情况

### 3. 学习问题解答
- 回答学生提出的学习问题
- 提供简单易懂的解答，结合乡村生活实际举例

### 4. 教师助手
- 为乡村教师提供教学支持
- 包括教案设计、教具制作、作业设计和课堂管理等功能

## 技术栈
- HTML5
- CSS3
- JavaScript (ES6+)
- ModelScope API

## 使用说明

### 1. 获取API密钥
访问 [ModelScope官方网站](https://www.modelscope.cn/my/myaccesstoken) 注册账号并获取API密钥。

### 2. 运行项目
在浏览器中打开 `index.html` 文件即可使用。

### 3. 功能使用
1. 输入ModelScope API密钥进行验证
2. 选择需要的功能模块
3. 根据提示输入相关信息
4. 点击对应按钮生成内容

## 开发指南

### 项目结构说明
- `index.html`: 页面结构和内容
- `css/styles.css`: 所有样式定义
- `js/main.js`: 所有交互逻辑

### 扩展开发
1. 添加新功能时，在对应目录下创建新文件
2. 样式修改请在 `css/styles.css` 中进行
3. 逻辑修改请在 `js/main.js` 中进行
4. 静态资源请放在 `assets/` 目录下

## 部署说明
将所有文件部署到Web服务器即可。
